# CDP — Customer Data Platform (NotNaked)

## What is this project?

This is an academic project to design and implement a Customer Data Platform (CDP) for NotNaked, a retail brand running on Shopify. The CDP serves as the central hub for all customer-related information — unifying data from multiple sources into a single, consistent customer profile.

## What are we actually learning here?

The CDP is the vehicle. The real goal is to learn and practice:

- **SDLC** — how software moves from requirements to a working system
- **Project Management** — how work gets scoped, broken into phases, tracked and delivered
- **Git Workflow** — how code and documentation are managed via branches, commits, issues and pull requests
- **Software Design** — data modeling, system modeling, architecture decisions
- **Business Logic** — workflows that map to real functional requirements

Reference: Ian Sommerville's *Software Engineering, 9th Edition*
